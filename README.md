
# [n](https://github.com/visionmedia/n) completion

## INSTALL

```
$ git clone git@github.com:hotoo/.n-completion.git ~/.n-completion
$ cd ~/.n-completion
$ make install
```

## USAGE

```
$ n li<tab>
$ n list

$ n st<tab>
$ n stable
```

## COMMANDS

Support auto complete commands:

* `n stable`
* `n latest`
* `n rm`
* `n ls`
* `n next`
* `n prev`
* `n use`
* `n bin`
* `n which`
* `n list`
* `n as`
